🔄 Notice de Synchronisation

Cartographie des Coffrets Électriques - Admin ↔ Lecture Seule

📖 Objectif de cette notice :
Vous apprendrez à transférer les données (coffrets, photos, liens) de la version Admin vers la version Lecture seule pour que votre équipe ait accès aux dernières mises à jour.

📋 Comprendre le principe

🔐 Version Admin

  • Vous ajoutez, modifiez, supprimez des coffrets
  • Vous uploadez des photos et ajoutez des liens
  • Les données sont stockées localement dans votre navigateur

👁️ Version Lecture Seule

  • Votre équipe consulte les coffrets
  • Les données sont aussi stockées localement dans leur navigateur
  • Pour voir vos modifications, il faut synchroniser
⚠️ Important :
Les deux cartes utilisent le localStorage du navigateur. Chaque navigateur a son propre stockage. Il faut donc exporter → importer pour transférer les données.

🎯 Méthode Recommandée : Export/Import

1

📤 Exporter depuis la carte Admin

Ouvrez la carte_admin.html et connectez-vous avec votre mot de passe.

En bas à gauche, cliquez sur le bouton :

💾 Exporter

Un fichier JSON est téléchargé, par exemple :

coffrets_electriques_hybrid_2026-02-08.json
2

📁 Sauvegarder le fichier JSON

Le fichier est dans vos Téléchargements.

Conservez-le précieusement :

  • ✅ Sauvegarde de toutes vos données
  • ✅ Restauration en cas de problème
  • ✅ Transfert vers d'autres navigateurs/ordinateurs
💡 Conseil : Créez un dossier "Sauvegardes Cartographie" et nommez vos fichiers avec la date, ex: 2026-02-08_coffrets.json
3

📤 Partager le fichier avec l'équipe

Plusieurs options :

  • SharePoint : Uploadez le JSON dans la même bibliothèque que les cartes
  • Email : Envoyez le fichier par email (si petit < 5 MB)
  • Drive partagé : Google Drive, OneDrive, Dropbox
  • Réseau local : Dossier partagé
4

📥 Importer dans la carte Lecture Seule

Chaque membre de l'équipe doit :

  1. Télécharger le fichier JSON
  2. Ouvrir carte_lecture_seule.html
  3. Cliquer sur le bouton : 📂 Importer
  4. Sélectionner le fichier JSON téléchargé
  5. Confirmer l'importation
✅ Synchronisation terminée !
L'équipe voit maintenant toutes vos modifications.

⚡ Méthode Avancée : Console Navigateur

⚠️ Pour utilisateurs avancés uniquement
Cette méthode nécessite d'utiliser la console du navigateur (F12).
1

📤 Exporter les données (Admin)

  1. Ouvrez carte_admin.html
  2. Appuyez sur F12 (outils développeur)
  3. Allez dans l'onglet Console
  4. Tapez cette commande et appuyez sur Entrée :
console.log(localStorage.getItem('coffrets_data_hybrid_v1'));
  1. Un long texte JSON s'affiche
  2. Clic droit sur le texte → Copier
  3. Collez dans un fichier texte et sauvegardez
2

📥 Importer les données (Lecture Seule)

  1. Ouvrez carte_lecture_seule.html
  2. Appuyez sur F12
  3. Allez dans l'onglet Console
  4. Tapez cette commande (remplacez VOTRE_JSON par les données copiées) :
localStorage.setItem('coffrets_data_hybrid_v1', 'VOTRE_JSON_ICI');
  1. Appuyez sur Entrée
  2. Rechargez la page (F5)
✅ Données synchronisées !

📅 Fréquence de Synchronisation

Fréquence Quand l'utiliser Avantages
Quotidienne Modifications fréquentes (plusieurs coffrets/jour) Équipe toujours à jour
Hebdomadaire Modifications régulières mais pas quotidiennes Bon équilibre temps/fraîcheur
Mensuelle Peu de modifications Moins de maintenance
À la demande Ajout d'un nouveau site ou gros changements Synchronisation ciblée

🔄 Workflow Recommandé

📋 Processus standard

  1. Lundi matin : Administrateur exporte les données de la semaine précédente
  2. Lundi 9h : Upload du JSON sur SharePoint
  3. Lundi 9h30 : Email à l'équipe : "Nouvelles données disponibles"
  4. Toute la semaine : Équipe travaille avec les données à jour
  5. Vendredi soir : Administrateur ajoute nouveaux coffrets (Admin)
  6. Répéter : Le lundi suivant

❓ FAQ - Questions Fréquentes

Q1 : Que se passe-t-il si je n'importe pas les données ?

R : La version Lecture Seule affichera les anciennes données. Aucun risque de perte, mais l'équipe ne verra pas vos modifications.

Q2 : Puis-je perdre mes données ?

R : Si vous exportez régulièrement (fichier JSON), non. Le fichier JSON est votre sauvegarde complète.

Q3 : L'import écrase-t-il les données existantes ?

R : Oui, l'import remplace complètement les données. Exportez avant d'importer si vous voulez conserver l'ancien état.

Q4 : Puis-je avoir plusieurs versions Admin ?

R : Oui, mais attention : chaque Admin a son propre stockage local. Utilisez Export/Import pour synchroniser entre Admins.

Q5 : La synchronisation fonctionne-t-elle sur smartphone ?

R : Oui ! Export et Import fonctionnent aussi sur mobile (iOS et Android).

💡 Bonnes Pratiques

✅ À FAIRE :
  • Exporter quotidiennement ou hebdomadairement (sauvegarde)
  • Nommer les fichiers avec la date (ex: 2026-02-08_coffrets.json)
  • Conserver les 3 dernières sauvegardes
  • Tester l'import sur votre propre carte Lecture Seule avant de partager
  • Informer l'équipe quand vous synchronisez
❌ À ÉVITER :
  • Ne jamais modifier manuellement le fichier JSON (risque de corruption)
  • Ne pas oublier d'exporter avant une réinstallation de Windows/navigateur
  • Ne pas partager le fichier carte_admin.html avec toute l'équipe (sécurité)
  • Ne pas importer un vieux JSON sans vérifier la date

🆘 En Cas de Problème

Problème : "L'import ne fonctionne pas"

Solutions :

  • Vérifiez que le fichier a bien l'extension .json
  • Vérifiez que le fichier n'est pas corrompu (ouvrez-le dans un éditeur de texte)
  • Essayez avec un autre navigateur
  • Videz le cache du navigateur (Ctrl+Shift+Suppr)

Problème : "J'ai perdu mes données"

Solutions :

  • Cherchez dans vos Téléchargements un ancien fichier JSON exporté
  • Vérifiez dans SharePoint si un JSON a été uploadé
  • Si vous avez utilisé un autre navigateur, ouvrez-le (les données sont peut-être là)
  • Contactez un membre de l'équipe qui a importé récemment (il peut ré-exporter)

📊 Récapitulatif

✅ Synchronisation Simple en 4 étapes :

  1. Carte Admin → Clic sur 💾 Exporter
  2. Partager le fichier JSON avec l'équipe (SharePoint, email, etc.)
  3. Carte Lecture Seule → Clic sur 📂 Importer
  4. Sélectionner le fichier JSON → Terminé ! 🎉
